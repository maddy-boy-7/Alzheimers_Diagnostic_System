# Alzheimers_Diagnostic_System
A comprehensive Alzheimer's prediction system using Clinical Data (Logistic Regression) and MRI Brain Scans (CNN/EfficientNet-B0). B.Tech Final Year Project.


## Introduction
Alzheimer's disease is a complex neurodegenerative disorder that affects millions of people worldwide. Early detection and prediction of Alzheimer's can lead to better management and treatment outcomes. This diagnostic system utilizes machine learning models to provide predictions about the likelihood of Alzheimer's disease.
